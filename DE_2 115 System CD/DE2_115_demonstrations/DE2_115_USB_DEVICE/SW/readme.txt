The C# example Template is used to display the template with the device connected
and the driver to which it has connected.

Template example support Visual studio 2008 and 2010. 

If you have Visual studio 2008 installed on your system , please use solution : \Template\Vs2008_sln\TemplateVS2008.sln to open the project. This solution will load the TemplateVS2008.csproj project file. 

If you have Visual studio 2010 installed on your system , please use solution : \Template\Vs2010_sln\TemplateVS2010.sln to open the project. This solution will load the TemplateVS2010.csproj project file.
